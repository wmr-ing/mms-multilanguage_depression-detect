Dataset: M-DEA

M-DEA is a dataset consisting of audio and text files , which is a multilingual dataset consisting of the DIAC, EATD-corpus and AVEC-2014 datasets.
Each sample in the dataset includes the original audio, translated text, and labels indicating “depressed” or “non-depressed”.
